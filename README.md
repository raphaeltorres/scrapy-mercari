# scrapy-mercari
Scrape mercari infinite scroll

# Installation
Clone Repo
 > git clone https://github.com/raphaeltorres/scrapy-mercari.git

 Create virtualenv
 > virtualenv -p python3 mercari

 Activate virtualenv
 > source mercari/bin/activate

 Install scrapy
 > pip install -r requirements.txt

Scrape Products
> $ scrapy crawl products
